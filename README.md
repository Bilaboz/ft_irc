# ft_irc

A simple IRC server written in C++98

## Compiling

```sh
git clone https://github.com/Bilaboz/ft_irc.git
cd ft_irc
make
```

## Usage

```sh
./ft_irc <port> [password]
```
Example: `./ft_irc 6667 securepassword`<br />
Note: the password is optional
